# Narrated explorable

Term I’ve used to describe media akin to [[Explorable explanations]] but which integrate the narrative experience into the interactive environment, instead of separating them as is usually done.

Examples:

  * [[Visualizing Quaternions]] by [[Grant Sanderson]] and Ben Eater
  * [[Introduction to Waves]]
  * [[Khan Academy’s computer programming lessons]]
    * scaled up: [[Scrimba]]
  * [[Cantor]], from me and my team at Khan Academy

See also [[Liqvid]], a framework for creating simple media of this kind.

Related: [[Dynamic mediums usually lack an authored time dimension]]

# How important is it to write your own spaced repetition memory prompts?

  * [[Prompt-writing supports reading comprehension]]
  * Apart from the general process effects on comprehension, prompt-writing probably produces more reliable long-term memory of the specific details in the prompts, via [[Elaborative encoding]] effects.

* * *

Nielsen, M. (2018). _Augmenting Long-term Memory_. <http://augmentingcognition.com/ltm.html>

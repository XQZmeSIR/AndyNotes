# Thought Saver

[[Spaced repetition memory system]] created by Spencer Greenburg and associates. Includes support for widgets so that it functions as a host for the [[Mnemonic medium]].

CEO: Flo Hinder

## Mnemonic medium implementation

Like the mnemonic medium, it offers support to authors via embedded iframes on web pages. The model seems a bit different. After I answered the first set of questions, it appears that all the prompts were added to my collection.

It looks like it makes no attempt to share progress between the iframes. In fact, I don’t think in-essay interactions are saved at all.

# To what extent does spaced repetition memory system practice constitute deliberate practice?

Do the review sessions of a [[Spaced repetition memory system]] constitute [[Deliberate practice]]?

  * Ways in which it is:
    * It’s explicitly about improving performance; it’s not inherently enjoyable
    * The exponential backoffs approximate focusing on your weaknesses
  * Ways in which it’s not:
    * Monitoring only approximates what an expert coach would do
    * It’s typically not all that effortful, per-se—feels more automatic. You know it or you don’t. Doesn’t really feel like challenging homeostasis: [[Skill development requires challenging homeostasis]]

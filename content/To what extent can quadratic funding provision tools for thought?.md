# To what extent can quadratic funding provision tools for thought?

[[Tools for thought]] are difficult to fund, in large part because [[Novel user interface ideas are public goods]]. Can [[Quadratic funding of public goods]] help?

This mechanism seems promising for funding implementation and maintenance stages of projects, but I have a few concerns:

  * [[The crowd probably can’t reliably assess early-stage research]]
  * [[Quadratic funding lacks incentives to compensate researchers for pre-proposal speculative work]]
